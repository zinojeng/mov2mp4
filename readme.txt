# MOV to MP4 Converter

## What
A command-line tool that converts MOV video files to MP4 format with customizable quality settings.

## Requirements
- Accept MOV files as input
- Convert to MP4 format using H.264 codec
- Support batch conversion of multiple files
- Allow quality/compression settings (low, medium, high)
- Display conversion progress
- Preserve original file metadata when possible
- Handle errors gracefully with clear error messages

## User Stories
- As a user, I want to convert a single MOV file to MP4
- As a user, I want to convert multiple MOV files at once
- As a user, I want to choose the output quality
- As a user, I want to see the progress of my conversion
- As a user, I want the converted files saved in the same directory or a specified output directory
